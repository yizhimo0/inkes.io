一. 用 hexo 框架搭建个人博客

下载 node.js 下载 git (具体步骤自己百度)

下载完成后_打开_ git bash

node -v
v12.16.1 (版本编号)

npm -v
6.13.4

到这里就安装 node.js 完成

开始安装 hexo 博客框架

 npm install -g cnpm --registry=https://registry.npm.taobao.org

可能需要一段时间 

cnpm install -g hexo-cli

hexo -v (验证)

hexo安装完成

创建文件夹

cd d:/
(更改存放地址)

mkdir blog

进入 bolg

cd d:blog/

生成一个博客

subo hexo init

....(可能会时间有点长)

查看目录生成的文件

ls -l

启动博客

hexo s

会得到 http://localhost:4000/

这里 ctrl +C 可以退出 创建文章

hexo n "我的第一篇博客文章"

ls (查看目录)

编辑文章

cd source/_posts/

ls -l

vim 我的第一篇博客文章.md

i (插入)

(markdown 来写)

##内容
---
内容
内容
---
内容
内容
---

按 ESC，左下角就可以进行输入 :w 保存但不退出 :wq 保存并退出 :q 退出 :q! 强制退出，不保存 :e! 放弃所有修改，从上次保存文件开始再编辑命令历史

注: 可以 vscode 等其他其他软件打开编辑 退回

cd ../..

清理

hexo clean

生成

hexo g

重新启动

hexo s

有刚才输入的内容,到这里hexo博客的搭建已经完成了
