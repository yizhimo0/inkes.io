hugo 博客搭建步骤

1.创建站点

hugo new site myblog
cd myblog

1.1创建文件

hugo new post/博客.md

2.下载主题

cd themes
git clone https://github.com/zwbetz-gh/cupper-hugo-theme.git cupper

hugo --theme=npq-hugo --baseUrl="https://yizhimo0.github.io/" --buildDrafts

hugo server --buildDrafts 

hugo server --theme=npq-hugo --buildDrafts

3.上传github

cd public
git init
git add .
git commit -m "第一次提交"
git remote add origin https://github.com/yizhimo0/yizhimo0.github.io.git
git pull origin master --allow-unrelated-histories
git push -u origin master

第二次上传 删除public文件避免和上传文件冲突

hugo server --theme=npq-hugo --buildDrafts
hugo --theme=npq-hugo --baseUrl="https://yizhimo0.github.io/"
